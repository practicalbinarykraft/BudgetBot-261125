Текущий UI очень ограниченный. Вот проблемы и улучшения:

❌ ПРОБЛЕМЫ ТЕКУЩЕГО UI
Показывает только 1 кошелёк → неудобно если их 5-10
Не видно разницу заранее → пользователь не понимает что произойдёт
Нет warning если разница большая (>10% = подозрительно)
Нет preview что будет создана транзакция "Неучтённые расходы"
Нет истории когда последний раз калибровали
✅ УЛУЧШЕННЫЙ UI
